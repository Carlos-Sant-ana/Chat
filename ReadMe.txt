Você precisa ter o Deno instalado para executar este aplicativo localmente.
1º Clonar o repositorio.
2º Vá para a raiz do projeto usando o terminal.
3º Run deno run --allow-net --allow-read server.js
4º Abra http: // localhost: 3000 / chat.html` no navegador
